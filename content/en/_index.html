PGA
The easiest way to
install and run Postgres 

        
            curl -sfL https://pga.sh/install | sh -
Read the Docs Star on GitHub

PGA offers a blazing-fast getting-started experience. It enables you to have multiple major and minor versions of Postgres running at the same time.

Installation and uninstallation are a one-liner, with no system dependencies required, managing your Postgres instances without compromising your existing system configuration.

Apart from the CLI interface, PGA allows you to define your Postgres clusters in YAML files. And has a REST API for integrations.

PGA is fully Open Source, distributed under the Apache 2 license.
